# Minimum Viable Product Demo APP

A basic log in and authentication web app.

## Installation

clone repo

cd into repo dir

inside of repo do the following commands

```bash
npm install && npm run client-install
```
## Built With

* [MongoDB]
* [ExpressJS]
* [ReactJS]
* [Redux]
* [NodeJS]
